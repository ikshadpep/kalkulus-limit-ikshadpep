Contoh Limit

1. Limit Fungsi

lim 𝑥 → 2 (2𝑥 + 3)

Substitusi 𝑥 = 2 langsung ke fungsi:

lim 𝑥 → 2 (2𝑥 + 3) = 2(2) + 3 = 7

2. Limit Tak Hingga

𝑓(𝑥) = 1/𝑥. Cari :

lim 𝑥 → ∞ 1/𝑥

Ketika 𝑥 menjadi sangat besar (𝑥 → ∞), nilai 1/𝑥 mendekati nol. Maka :

lim 𝑥 → ∞ 1/𝑥 = 0

3. Limit Trigonometri

lim 𝑥 → 0 sin(2𝑥)/𝑥

Penyelesaian : Gunakan sifat limit :

lim 𝑥 → 0 sin(2𝑥)/𝑥 = lim 𝑥 → 0 sin(2𝑥)/2𝑥 . 2

Diketahui bahwa lim 𝑥 → 0 sin(2𝑥)/2𝑥 = 1, maka :

lim 𝑥 → 0 sin(2𝑥)/𝑥 = 1 . 2 = 2

4. Limit Kanan Kiri

f(𝑥) = {𝑥 + 1, jika 𝑥 < 2

𝑓(𝑥) = {2𝑥 - 3, jika 𝑥 ≥ 2

Untuk menentukan limit 𝑥 → 2 :

Limit Kiri (𝑥 → 2-) : ketika 𝑥 < 2, gunakan bagian fungsi 𝑥 + 1.

lim 𝑥 → 2- f(𝑥) = 2 + 1 = 3

Limit Kanan (𝑥 → 2+) : Ketika 𝑥 ≥ 2, gunakan bagian fungsi 2𝑥 - 3.

lim 𝑥 → 2+ f(𝑥) = 2(2) - 3 = 1

Jika Limit kanan dan Limit Kiri tidak sama, maka limit 𝑥 → 2 Tidak ada.
